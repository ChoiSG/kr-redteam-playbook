# Pass-the-Hash

패스 더 해시 (Pass-the-Hash, PtH, PTH) 기법은 윈도우 NTLM 인증을 사용할 때 평문 비밀번호를 사용하지 않고, 그 비밀번호의 NT 해시를 사용해 인증하는 기법을 일컫는다.&#x20;

패스 더 해시 기법의 가장 큰 오해는 바로 이것이 공격(attack)이라는 것이다. 패스 더 해시는 공격도 아니고, 특정 프로토콜을 악용(abuse)하는 것도 아니다. NTLM 인증 프로토콜의 취약점(vulnerability)는 더더욱 아니다. 따라서 내부망 모의해킹을 진행할 때도 이 PTH 를 취약점으로 적어놓는 실수를 하면 안된다.&#x20;

패스 더 해시는 그냥 평범하게 NTLM 인증을 활용하는 기법 중 하나다. 만들어놓은 프로토콜을, 의도한대로 평범하게 사용하는 것이다. 그렇다면 NTLM 인증은 어떻게 진행될까?&#x20;
