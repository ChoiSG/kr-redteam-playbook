# CreateRemoteThread

CreateRemoteThread 인젝션은 원격 프로세스에 쓰레드를 만들어 공격자의 쉘코드를 실행시키는 전통적인 프로세스 인젝션 방법 중 하나다.&#x20;

### 실행 단계&#x20;

1. OpenProcess - 원격 프로세스의 핸들을 얻는다&#x20;
2. VirtualAllocEx - 원격 프로세스에 메모리를 할당한다&#x20;
3. WriteProcessMemory - 할당된 메모리에 쉘코드를 쓴다
4. VirtualProtectEx - 할당된 메모리 권한을 RX로 바꾼다&#x20;
5. 원격 프로세스에 쓰레드를 생성해 해당 쓰레드가 쉘코드를 실행하게끔 한다&#x20;
